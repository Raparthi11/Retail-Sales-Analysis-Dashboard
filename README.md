# Retail Sales Performance Dashboard

## Project Overview
This project presents an **interactive Retail Sales Performance Dashboard** built using Tableau to analyze sales data and uncover key business insights.

The dataset was first cleaned and transformed using **Excel**, where additional KPI-related columns were created. The processed dataset was then used in **Tableau** to build an interactive dashboard with multiple visualizations and filters to support data-driven decision-making.

---

## Key Metrics
- **Total Sales:** 2.30M
- **Total Profit:** 286.40K
- **Total Quantity Sold:** 37.87K
- **Average Profit Margin:** 12.06%

---

## Dashboard Insights
The dashboard helps analyze:

- Sales trends over time
- Regional sales performance
- Category-wise sales distribution
- Profit by sub-category
- Top customers contributing to revenue

Interactive filters allow users to explore the data by:
- Region
- Category
- Year
